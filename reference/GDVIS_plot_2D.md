# Plot GDVIS 2D plot

This function plots a GDVIS visualization based on GDVIS parameters.

## Usage

``` r
GDVIS_plot_2D(
  input_triangle_parameters,
  x_lower = NULL,
  x_upper = NULL,
  y_lower = NULL,
  y_upper = NULL
)
```

## Arguments

- input_triangle_parameters:

  the path to the RData GDVIS output, which ends with
  2D.triangle_parameters.RData

- x_lower:

  set lower value for the x axis

- x_upper:

  set upper value for the x axis

- y_lower:

  set lower value for the y asis

- y_upper:

  set upper value for the y axis
